# Pix4Dauto

## Description
Program to create Pix4D projects automatically using a predefined template. After a day of filled work with multiple missions and in a time sensible project, simply organize the mission's images and launch projectAuto.bat. No need to wait next to the computer, the results will be in the next day.

## Usage
- Sign in to Pix4D Mapper with your license (must not be trial version to use command line)
- Import folders of filtered images in 'raw' folder contained in base directory
- Set base directory and template in settings.py
- Run test.py
